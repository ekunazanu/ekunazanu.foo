## Attribution

The design of this site is based on [Tom MacWright](https://macwright.com)'s beautiful and minimal site. The main typeface of this site is [Inter](https://rsms.me/inter/), and the monospace font is [JetBrains Mono](https://www.jetbrains.com/lp/mono/). The serifs are [Playfair Display](https://www.forthehearts.net/playfair2_0/). I wanted to go with system fonts, but I like the look of the typefaces.

I try to use my own drawings and media files whenever I can. But if not, it is sourced from either [unsplash](https://unsplash.com/), [pixabay](https://pixabay.com/) or [pexels](https://www.pexels.com/). The entire site is built using [Zola](https://www.getzola.org/).
