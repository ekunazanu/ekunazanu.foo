Attribution

The design of this site is based on Tom MacWright’s beautiful and minimal site. The main typeface of this site is Inter, and the monospace font is JetBrains Mono. The serifs are Playfair Display. I wanted to go with system fonts, but I like the look of the typefaces.

I try to use my own drawings and media files whenever I can. But if not, it is sourced from either unsplash, pixabay or pexels. The entire site is built using Zola.
